# Agents

GuideAtlas will use autonomous agents to accelerate platform growth.

## Planned Agents (FUTURE)

### Guide Discovery Agent (FUTURE)
Finds local guides online and invites them to the platform.

### SEO Content Agent (FUTURE)
Generates destination-focused content to improve discovery and SEO reach.

### Tour Generator Agent (FUTURE)
Creates curated tour drafts based on city context and traveler intent.

### Social Media Agent (FUTURE)
Produces and schedules social content to promote guides and tours.

## Operating Principle
Agents must read docs in `docs/` before changing code.
